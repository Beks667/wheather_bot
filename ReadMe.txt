Создать новый файл в вашей папке в формате питона .py (Например: tg_bot.py)

Открыть терминал (Win + R и прописать CMD)

В терминале перейти в вашу директорию(папку) используя команду "cd"(например: 'cd Desktop/MyFolder/')

Когда вы уже непосредственно находитесь в вашей папке вам нужно скачать библиотеку aiogram

введите в терминале команду pip install aiogram для установки библиотеки 






Для работы с телеграммом нам для начала нужно Зарегистрировать нашего бота и получить персональный токен, для этого:

Откройте Телеграмм на ноутбуке веб версию или Десктопное приложение 
https://web.telegram.org/k/

Возьмите телефон и отсканируйте QR code

В поисковой строке напишите @BotFather

откройте его и начните создавать вашего бота,для это в его меню найдите команду /newbot 

Следуйте правилам регистрации 

После успешной регистрации вам в чат придет большое сообщение с токеном сохраните его 



для того что бы мы могли отслеживать погоду нам надо зарегистрироваться на сайте и точно также получить персональный токен 

перейдите на сайт https://openweathermap.org/

зарегистрируйтесь на этом сайте 

нажав на свой профиль передите в отдел My API keys

сохраните токен
